## Features

- Classic Snake gameplay
- Score tracking
- High score system with local save
- Main menu and game over screen
- Smooth controls

## Dependencies

- [Raylib](https://github.com/raysan5/raylib) — automatically fetched via CMake FetchContent

No manual installation required.

## Build & Run

**Requirements:** g++ or clang++, CMake 3.10+, Git

```bash
# Clone the repository
git clone https://github.com/XeradSE/Snake.git
cd Snake

# Configure and build
mkdir build
cd build
cmake ..
cmake --build .

# Run
./Snake
```

## Controls

| Key | Action |
|-----|--------|
| ↑ ↓ ← → | Move |
| Enter | Start / Restart |
